EINLEITUNG
----------
TextPad wurde konzipiert, um die Leistung und Funktionalität für die
strengsten Textbearbeitungsanforderungen zu erfüllen. Es läuft unter
Windows und ist in 16 und 32-bit Versionen erhältlich. Große Dateien
können von beiden Ausführungen bearbeitet werden, wählen Sie die
Version, die am besten auf Ihrem Computer läuft. Die 32-Bit Version
kann Dateien bis hin zum virtuellen Speicher bearbeiten (für genaue
Angaben schauen Sie bitte  unter Spezifikationen in der on-line Hilfe
nach), und arbeitet mit Windows 95, NT und Windows 3.1 mit der Win32s
Erweiterung.

Wenn Sie mehr erfahren möchten, installieren Sie TextPad. Bitte folgen
Sie der Installationsanweisung und lesen dann bitte die Einleitung der
on-line Hilfe Datei.

INSTALLATION:
--------------
    TextPad verfügt über ein Installationsprogramm, welches es auf die
    von Ihnen genutzte Version von Windows abstimmt. Einige Merkmale
    werden nicht einwandfrei funktionieren (einschließlich des
    Deinstallationsprogrammes), wenn Sie Textpad nicht
    vorschriftsmäßig installieren. Sie wurden gewarnt!

    Um TextPad aus einer ZIP Datei zu installieren:

      1) Dekomprimieren aller Dateien in ein leeres, temporäres
         Verzeichnis, mit PKUNZIP 2.04g.
      2) Führen Sie SETUP.EXE aus und folgen Sie den Anweisungen auf
         dem Bildschirm.
      3) Achten Sie darauf, daß Sie TextPad in einem anderen als dem
         unter (1) benutzten Verzeichnis installieren.
      4) Löschen Sie die dekomprimierten Dateien aus (1).

    WICHTIGE MITTEILUNGEN:

    + Wenn Sie Dateierweiterungen mit TextPad verknüpfen möchten,
      erreichen Sie dies, indem Sie die Assoziations-Tabs der
      Vorzugsdialogbox verwenden. Damit stellen Sie sicher, daß
      TextPad mit den richtigen Parametern gestartet wird und
      ermöglicht dem Deinstallationsprogramm, TextPad anschließend
      vollständig zu löschen.

    + Wenn Sie die 32-Bit Version mit Windows 3.1 oder 3.11 benutzen
      möchten, müssen Sie zunächst Win32s 1.30.172 installieren. Dadurch
      werden auch die 16-Bit OLE2 Dateien installiert. Wenn Sie
      Zugriff auf das Internet haben, können Sie es auf folgender
      Adresse kopieren:

      http://premium.microsoft.com/support/win32dev/faq/win32s/all.asp

      Benutzer von CompuServe können Win32s vom MSWIN Forum als
      PW1118.EXE kopieren.
