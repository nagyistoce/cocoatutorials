EXIV2 BVT AUTOMATION

Exiv2 Build Verification Test Automation is to build/test the Exiv2 automatically.
It will check the latest svn revision of Exiv2 and run build/test if it's necessary.
The cronjob is used to check the svn revision every hours.

Current Status

The Last Build: %LastBuild. View log at here.

The Last BVT Run: %LastBVT.

Actions

Check latest SVN Revision and run BVT.

Reset BVT (Remove the old log file, allow to run BVT for the build %LastBuild again).
